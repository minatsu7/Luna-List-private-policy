隱私權政策 (Privacy Policy)
生效日期：2024年1月

Luna List - 月亮清單 (以下簡稱「本應用程式」) 非常重視您的隱私權。本隱私權政策將說明我們如何收集、使用及保護您的個人資訊。請您詳細閱讀以下內容。

1. 資料收集與使用 (Data Collection and Use)
1.1 個人資料
本應用程式主要為 本機端 (Offline-first) 應用程式。我們不會將您的待辦事項、日記內容、心情紀錄或願望清單上傳至任何外部伺服器。

所有的使用者產生內容 (User Generated Content) 皆儲存於您的裝置內部空間。
若您刪除本應用程式，儲存於裝置內的資料將會一併被刪除（除非您有自行進行備份）。
1.2 應用程式權限
為了提供完整功能，本應用程式可能會要求以下權限：

通知權限 (Notifications)：用於發送新月提醒。您可以隨時在裝置設定中關閉此權限。
網路存取 (Network Access)：僅用於應用程式內購買 (In-App Purchase) 的驗證與溝通，以及極少量的非個人化數據（如 Crashlytics 崩潰報告，若有的話）以改善 App 穩定性。

2.1 應用程式內購買 (In-App Purchases)
我們使用 Apple App Store 與 Google Play Store 的支付系統來處理任何應用程式內的購買行為（如解鎖主題）。

本應用程式不會收集或儲存您的信用卡號碼或銀行帳戶資訊。所有的交易皆由 Apple 或 Google 的安全支付平台處理。

3. 資料安全 (Data Security)
我們致力於保護您的資料安全。由於您的個人數據主要儲存於您的本機裝置，請確保您的裝置設有密碼保護，以防止未經授權的存取。

4. 兒童隱私 (Children’s Privacy)
本應用程式不以 13 歲以下兒童為目標客群，亦不會有意收集兒童的個人資訊。
